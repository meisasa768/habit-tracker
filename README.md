# 📖 学習記録帳

公務員試験・就活対策の習慣トラッカーアプリです。

## 機能

- **今日の記録** — 6項目のチェック＋達成数の入力
- **カレンダー** — 月ごとの達成状況を色で確認・過去日の編集
- **グラフ** — 過去30日の棒グラフ・連続streak・全項目ヒートマップ

## 記録する習慣

| アイコン | 項目 |
|---------|------|
| 🌐 | 英文を読む |
| 📰 | 時事学習 |
| 💡 | ITビジネスニュース |
| 🏛️ | 県庁だより |
| 🎤 | 県職員インタビュー |
| 🔢 | 数的処理1問 |

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## ビルド

```bash
npm run build
```

`dist/` フォルダが生成されます。GitHub Pages や Vercel にデプロイ可能です。

## データ保存

記録はブラウザの `localStorage` に保存されます。同じブラウザであれば次回以降も引き継がれます。
